# Vue3 商品交易系统 java771

本项目是一个基于Vue3的简单商品交易系统，主要包括商城后台和用户管理等功能。以下为项目的详细介绍。

## 技术栈

- **项目框架**：基于 `Vue3` 和 `Vite` 搭建
- **路由管理**：使用 `Vue Router 4+`
- **数据请求**：采用 `axios` 进行数据请求模拟
- **UI组件库**：使用 `Element Plus`

## 功能模块

- 用户登录注册
- 购物车管理
- 后台管理
- 商品管理

## 系统角色

- 普通用户：可进行商品浏览、购买、添加至购物车等操作
- 管理员：负责商品管理、后台管理等操作

## 运行环境

- **开发环境**：推荐使用 `Node.js` 和 `npm`
- **浏览器支持**：请使用最新版本的现代浏览器

## 目录结构

```
src/
|-- assets/
|-- components/
|-- views/
|-- router/
|-- store/
|-- App.vue
|-- main.js
```

## 核心亮点

- **纯前端实现**：本系统为纯前端无后台、无数据库的Vue3项目作业
- **Vue3 setup语法糖**：采用Vue3的setup语法糖写法，使得代码更加简洁和易于维护

## 部署步骤

1. 克隆项目代码到本地
2. 安装依赖：`npm install`
3. 运行开发环境：`npm run dev`
4. 打包项目：`npm run build`

请注意，本项目的部署和运行仅涉及前端部分，不涉及后端及数据库部署。
